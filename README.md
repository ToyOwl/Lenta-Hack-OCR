# Lenta-Hack-OCR
Lenta OCR Price Tag Pipeline:Проект реализует end-to-end пайплайн распознавания ценников и ценниковых лент в формате розничной сети «Лента»: входное изображение проходит нормализацию, определение структуры, классификацию шаблона, OCR, извлечение цены/товара/кодов и формирование итоговой карточки товара в JSON/TSV.
